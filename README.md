# Мини-википедия
Приложение, с помощью которого можно искать текстовую информацию в Wikipedia.
## Для запуска сервера используй команду `npm run server`, для запуска клиента - `npm run client`.

На *главной странице* находится строка поиска и кнопка. При нажатии на кнопку на страницу выводится краткая информации и ссылка на википедию.
### Дополнительные фишки:
- На вкладке *"История"* выводятся предыдущие запросы с использованием сортировки по времени.
- В *шапке* приложения есть переключатель на темную тему.

Информация о клиенте хранится в cookies. Запросы хранятся в псевдорепозитории на сервере, без использования базы данных.

Для создания клиентской части использовался React. Серверная часть написана с помощью Express. Сборка осуществляется с помощью Webpack.
